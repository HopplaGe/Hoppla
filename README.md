## This is a [Hoppla.Ge](https://hoppla.ge/) project.

## Installation

    ```bash
    $ git clone
    $ cd hoppla
    $ npm install
    ```
## Running the app

    ```bash
    # development
    $ npm run dev

    # production mode
    $ npm run build
    $ npm run start
    ```

## Test

    ```bash
    # unit tests
    $ npm run test

    # e2e tests
    $ npm run test:e2e

    # test coverage
    $ npm run test:cov
    ```
## Docker

    ```bash
    # development
    $ docker-compose up -d --build

    # production mode
    $ docker-compose -f docker-compose.prod.yml up -d --build
    ```
